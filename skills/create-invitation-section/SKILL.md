# Create Invitation Section

## Goal
Membuat atau menyempurnakan section undangan yang terasa seperti bagian dari User Interface Spotify App, dengan nuansa modern, dark theme, sleek, dan mobile-first.

## Requirements
Saat membuat section:
1. Pahami tujuan section terlebih dahulu (apakah ini "Album Art", "Artist Profile", atau "Tour Dates")
2. Gunakan hierarchy visual yang jelas, mengikuti tipografi aplikasi modern.
3. Tambahkan entrance animation dengan Framer Motion (fade in/slide up).
4. Tambahkan minimal satu elemen interaksi UI:
   - hover effect pada card
   - active state pada tombol
   - smooth layout transition
5. Pastikan layout mobile-first (sangat penting untuk Spotify UI).
6. Pastikan warna dan font mengikuti style project (dark mode `#121212`, `#181818`, aksen `#1DB954`, dan text `#FFFFFF`, `#B3B3B3`).
7. Hindari desain statis bertema vintage/klasik.

## Output Standard
Section yang dibuat harus:
- siap dipakai di React
- memakai Tailwind untuk styling
- memakai Framer Motion untuk animasi
- terasa seperti aplikasi native, modern, dark, dan clean
