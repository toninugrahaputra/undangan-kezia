# Create Invitation Section

## Goal
Membuat atau menyempurnakan section undangan yang terasa premium, elegan, hidup, dan mobile-first.

## Requirements
Saat membuat section:
1. Pahami tujuan section terlebih dahulu
2. Gunakan hierarchy visual yang jelas
3. Tambahkan entrance animation dengan Framer Motion
4. Tambahkan minimal satu elemen motion pendukung:
   - parallax
   - floating ornament
   - text reveal
   - smooth transition
5. Pastikan layout mobile-first
6. Pastikan warna dan font mengikuti style project
7. Hindari desain yang terasa statis atau template-like

## Output Standard
Section yang dibuat harus:
- siap dipakai di React
- memakai Tailwind untuk styling
- memakai Framer Motion untuk animasi
- terasa cinematic, elegant, dan premium
