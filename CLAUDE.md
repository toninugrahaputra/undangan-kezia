Project ini adalah website undangan digital premium berbasis React + Tailwind + Framer Motion.

Standar desain wajib:

Gunakan desain modern, cinematic, elegant, dan smooth animation.

Selalu gunakan:
- scroll animation
- parallax background
- text reveal animation
- floating elements
- smooth transition antar section
- confetti effect saat buka undangan
- responsive mobile-first layout

Gunakan komposisi warna:
soft gold, cream, white, pastel romantic

Font style:
serif elegan untuk heading
sans-serif clean untuk body

Setiap section wajib punya animation entrance.

Gunakan Framer Motion untuk semua animasi.

Jangan buat desain statis.
Semua UI harus terasa hidup dan premium.
